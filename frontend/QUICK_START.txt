┌──────────────────────────────────────────────────────────────────┐
│  🎯 HOW IT WORKS                                                 │
└──────────────────────────────────────────────────────────────────┘

  1. User clicks "Get My Location"
  2. Browser requests geolocation permission
  3. User grants → Lat/Lng stored
  4. Map centers on user location
  5. Blue marker appears
  6. Overpass API query sent:
     → Search within 5km radius
     → Query OSM for seed shops or markets
  7. Results received and normalized
  8. Distance calculated for each place
  9. Sorted by distance (closest first)
  10. Green/Orange markers placed on map
  11. List shows below with all details
  12. Click "Get Directions" → Opens Google Maps

    • name contains: mandi, market, wholesale, sabzi

┌──────────────────────────────────────────────────────────────────┐
│  ⚠️ KNOWN LIMITATIONS                                           │
└──────────────────────────────────────────────────────────────────┘

  Data Quality:
    • OSM data varies by region
    • Some areas have limited data
    • Rural areas may have fewer results
    • Urban areas usually well-mapped
  
  API Speed:
    • Overpass API is public, can be slow
    • Usually 2-5 seconds response time
    • During peak hours may be slower
    • No rate limits for fair use
  
  Solutions:
    • Cache results for 10 minutes
    • Show "Searching..." state
    • Handle empty results gracefully

┌──────────────────────────────────────────────────────────────────┐
│  🐛 TROUBLESHOOTING                                              │
└──────────────────────────────────────────────────────────────────┘

  ❌ Map not showing?
     → Check internet connection
     → Map tiles load from CDN
     → Check browser console
  
  ❌ Markers look wrong?
     → Leaflet CSS loaded?
     → Check icon CDN links
     → Clear browser cache
  
  ❌ No results found?
     → OSM data may be limited
     → Try different location
     → Check Overpass API status
  
  ❌ Location not working?
     → Must be HTTPS or localhost
     → Allow location permission
     → Check browser settings
